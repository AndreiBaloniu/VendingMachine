# VendingMachine
**Aplicatie tip consola in C++**

Am creat o aplicatie de tip consola in C++ in cadrul laboratoarelor de OOP. Timp de 3 saptamani am primit cate o problema si de a implementa diferite task-uri in timpul laboratoarelor dar si acasa. Am avut de facut un automat. Ca si entitati in proiect avem produsele si valuta (monede/ bancnote) pe care le validam. Citim din fisier banii si produsele din automat. Un produs are cod, nume si pret, iar valuta - tipul (moneda/ bancnota) si valoarea acesteia. Folosim cate un repository pentru fiecare entitate.  Am respectat principiile OOP precum incapsularea, implementarea dinamica, mostenirea si polimorfismul.  Aplicatia are un meniu pentru produse si valuta, de a cumpara produse si a le vizualiza. Automatul poate oferi rest cand cumparam ceva si platim cu o bancnota mare.
Exemple de functionalitati ce pot fi gasite sunt:

1. Adauga produs.
2. Sterge produs.
3. Update produs.
4. Afisare produse.
5. Search produs.
6. Numar produse.
7. Adauga mai multe produse de acelasi tip.
